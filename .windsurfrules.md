# Expected Behavior

When responsing to a prompt:
- Only make changes relevant to what has been requested.
- If there is a better approach than the prompt suggests, then ask if we should that that approach.
- In most cases, changes should be made iteratively, and they should be as small as possible.

# Running python and related commands
!! This is very important, do not forget this !!

When running python or pip or any other python commands, be sure to use the venv/bin/ version of the command. You can't be certain the venv is activated, so just play it save and use those versions explicitly. It also assures me that the right ones are being run.
